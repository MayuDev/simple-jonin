Features
Don't Know A Command?! • Jonin has ways to help you out, some commands list the commands and their functions-//help-and other commands help you understand how to use the bot-//explain [COMMAND].

API's • Jonin has a library of API's, New API's are added constantly and old API's are improved upon if a better one is found.

Used a command wrong? • Jonin has detailed instructions for most commands if there is a chance that the command could be misunderstood.

New commands • New commands can be requested by users and will be added if approved by a dev.
